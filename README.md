## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd synergy408todo
```
2.  Установите зависимости для бэкенда: `npm install`
3.  Установите зависимости для фронтенда: `cd client && npm install`
4. Откройте http://localhost:5173 в браузере
