# "To-Do List SITE "

Краткое описание: веб-приложение для управления личными задачами с поддержкой авторизации и постоянного хранения данных.

## Цели проекта

*   Изучить работу в команде с использованием Git/GitHub.
*   Освоить полный цикл разработки веб-приложения.
*   Применить на практике знания по [HTML, CSS и JavaScript].
*   Отобразить сайт

## Стек технологий

*   **Фронтенд**: HTML, CSS3, JavaScript
*   **Бэкенд**: Node.js, Express.js
*   **База данных**: SQLite (локальная, файловая)
*   **Авторизация**: Email + пароль (хеширование через `bcrypt`, токены через `jsonwebtoken`)
*   **Другие инструменты:** Git, GitHub.

## Основные возможности

*   ✅ Регистрация и вход по email и паролю
*   ✅ Хранение задач в **настоящей базе данных** (SQLite), а не в `localStorage`
*   ✅ Задачи привязаны к пользователю (данные не пересекаются между аккаунтами)
*   ✅ Автоматическое сохранение — изменения фиксируются без кнопки «Сохранить»
*   ✅ Фильтрация задач: «Все», «Активные», «Завершённые»
*   ✅ Удаление задач и очистка завершённых
*   ✅ Прогресс выполнения и статистика
*   ✅ Адаптивный дизайн и приятный UI

## Как запустить локально

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd synergy408todo
```
2.  Установите зависимости бэкенда
```bash
cd backend
npm install
```
`Сервер будет доступен на http://localhost:3000`

4.  Запустите сервер:
```bash
node server.js
```
5.  Откройте фронтенд:
`Перейдите в папку frontend`;
`Откройте файл index.html в браузере (двойным кликом или через локальный сервер)`.
